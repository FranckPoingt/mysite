My Beer Cellar (mybeercellar.org)

My Beer Cellar is a project started with a classmate based on a personal idea.

We started developing it as the final project of our AFPA courses. It is made using JSF with Primefaces and use SQL Server for Database.
JSF was imposed by the formation center and isn't what I would have personally use.

That's why I plan on rebuilding it using React / Spring and SQL Server.
I also plan on building a mobile app for it using Flutter. I already bought the domain names for it (French and Engligh).

The purpose of this app is to allow the user to keep track of the beers he bought and store in his cellar. More functionalities will come later on.
